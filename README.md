# Devs in the shed public website.

## Requirments:
- Python 3

## How to run:
cd to the folder of the cloned code and create a virtual env
```
virtualenv venv
```
Activate the virtual env
```
source venv/bin/activate
````
Start the app with Python
```
python app.py
```

Then browse to:  http://127.0.0.1:8080
